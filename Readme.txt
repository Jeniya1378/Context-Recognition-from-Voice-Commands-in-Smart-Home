Installation guide:
1. Install Python 
2. Go to any python IDE (example: Visual Code) and run "pip install -r requirements.txt"
Or, From command prompt run "pip install -r requirements.txt"
3. In any python IDE, run main.py 
Or, From command prompt run "python main.py"


Input: In the trained model, input any text to detect context
Output: Displays top 6 possible contexts. 


Notes: The accuracy might be different depending on environment.
In order to generate the same result it is recommended to install in a new/virtual environment.
